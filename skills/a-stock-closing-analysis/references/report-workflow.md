# 收盘日报生成流程

> 核心原则：**有数据、有分析、有信号**。
> 数据保留（北向多少钱、PE百分位），但配上解读让普通人看得懂。
> 多角度交叉验证：板块热度、聪明钱动向、异常信号。

## 数据采集链

```

│    e. 💰 聪明钱（北向+融资+龙虎榜机构）                    │

│    g. ⚡ 今日异常（现象描述）                              │

```

## 行情判断逻辑

| 判断条件 | 行情类型 | 开场白方向 |
|:---------|:---------|:----------|
| 涨跌比 > 2:1 且指数全红 | 🔥 普涨 | "今天全线飘红…" |
| 涨跌比 < 1:2 且指数全绿 | 🌧️ 普跌 | "今天整体回调…" |
| 指数有红有绿 | ⚡ 分化 | "今天板块分化明显…" |
| 指数 ±0.5% 以内 | ➖ 震荡 | "今天窄幅震荡…" |
| 涨停>50 或 指数涨>2% | 🚀 强势 | "今天多头强势…" |
| 跌停>50 或 指数跌>2% | ⚠️ 弱势 | "今天空头占优…" |

## 聪明钱分析规则

三个信号交叉验证：

```
北向资金  融资融券   龙虎榜机构    →  结论
──────────────────────────────────────
  买        买          买        →  🔥 内外资一致看多，强信号
  买        买          卖        →  ⚠️ 内外资看多但机构在卖
  卖        卖          卖        →  🔻 一致看空，警惕
  买        卖          买        →  🤔 外资+机构看多，散户在撤
  卖        买          买        →  🤔 机构抄底，外资在撤
```

**报告中的写法：**
```
✅ 正确：
  🌊 北向资金 +28.5亿（连续3日买入）
  📈 融资融券 -12亿（杠杆资金在撤退）
  🏦 龙虎榜机构净买入 5.2亿（机构在抄底）
  → 外资+机构看多，但杠杆资金偏谨慎，市场有分歧

✅ 也可以简洁版：
  🌊 北向 +28.5亿 ｜ 📈 两融 -12亿 ｜ 🏦 机构 +5.2亿
```

## 板块热度分析

用 `mx-xuangu` 查今日行业板块涨跌排名：

```bash
python ~/.hermes/skills/mx-xuangu/mx_xuangu.py "今日涨幅最大的5个申万行业"
python ~/.hermes/skills/mx-xuangu/mx_xuangu.py "今日跌幅最大的5个申万行业"
```

**报告中的写法：**
```
🔥 板块热力图
  ↑ 银行 +1.2%  房地产 +0.8%  煤炭 +0.5%  —— 防御板块领涨
  ↓ 计算机 -3.1%  电子 -2.8%  传媒 -2.5%  —— 科技板块回调

  → 资金往防御板块跑，说明市场情绪偏保守
```

## 估值体检规则

同时查三个核心指数的 PE 分位，形成全市场估值全景：

### lixinger 调用

```bash
TOKEN=$(python3 -c "import json;print(json.load(open('$HOME/.hermes/skills/lixinger/token.json'))['token'])")

# 沪深300 PE + 近3年分位
curl -s -X POST "https://open.lixinger.com/api/cn/index/fundamental" \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"stockCodes\":[\"000300\"],\"date\":\"$DATE\",\"metricsList\":[\"pe_ttm\",\"pe_ttm.y3.cvpos\"]}"

# 创业板指
curl -s -X POST "https://open.lixinger.com/api/cn/index/fundamental" \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"stockCodes\":[\"399006\"],\"date\":\"$DATE\",\"metricsList\":[\"pe_ttm\",\"pe_ttm.y3.cvpos\"]}"

# 科创50
curl -s -X POST "https://open.lixinger.com/api/cn/index/fundamental" \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"stockCodes\":[\"000688\"],\"date\":\"$DATE\",\"metricsList\":[\"pe_ttm\",\"pe_ttm.y3.cvpos\"]}"
```

### 分位解读

| PE 分位 | 人话 |
|:-------|:-----|
| 0-20% | 比过去3年80%的时间都便宜 |
| 20-40% | 低于历史平均，偏便宜 |
| 40-60% | 处于历史中间位置，价格合理 |
| 60-80% | 高于历史平均，偏贵 |
| 80-100% | 比过去3年80%的时间都贵 |

### 报告写法

```
📊 估值体检
  沪深300 PE 12.3x（近3年25%分位，偏便宜）
  创业板 PE 35.1x（近3年18%分位，比过去3年80%时间都便宜）
  科创50 PE 48.2x（近3年55%分位，价格合理）
```

## 异常信号检测

### 大宗交易

```bash
curl -s -X POST "https://open.lixinger.com/api/cn/company/block-deal" \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"date\":\"$DATE\"}"
```

如果当日有大宗交易，检查是否有金额异常大的（>1亿），或者折价/溢价率异常的。

### 限售解禁

```bash
curl -s -X POST "https://open.lixinger.com/api/cn/company/hot/elr" \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"stockCodes\":[\"000300\"]}"
```

如果有大规模解禁（>50亿），在"需要关注"中提醒。

### 报告写法

```
⚡ 今日异常
  🔔 银行板块集体上涨 +1.2%，但北向资金在卖出银行股 3.2亿 → 可能是内资在拉
  🔔 XX股份出现大宗交易 2.3亿，折价 8% → 需关注后续走势
```

## 小结规则

每期必须回答两个问题：
1. **今天怎么看？** — 一句话总结今天的核心特征
2. **明天注意什么？** — 一个具体的风险点或机会点

```
📝 今天跌停65家偏多，但估值处于历史低位，属于价格回调而非基本面恶化。
⚠️ 北向资金连续3日买入银行板块，如果明天继续，可能是风格切换信号。
```
