# 🌤️ 世界杯天气采集工作流

> 创建日期: 2026-06-06
> 目的: 每场比赛前采集当地实时天气预报，评估天气对比赛的影响

## 为什么天气重要

2026世界杯在美国/加拿大/墨西哥举办，跨越3个气候带：

| 气候类型 | 代表城市 | 风险 |
|---------|---------|------|
| 🏜️ 沙漠/半干旱酷热 | 蒙特雷、达拉斯(室外)、休斯顿(室外) | 35°C+，中暑风险 |
| 🌴 亚热带湿热 | 迈阿密、休斯顿 | 32°C+湿度>80%，体感40°C+ |
| 🌲 温带/海洋性 | 西雅图、温哥华、旧金山、波士顿 | 舒适15-25°C |
| 🏔️ 高原 | 墨西哥城(2200m)、瓜达拉哈拉(1560m) | 缺氧+午后雷雨 |
| ❄️ 可控环境 | 亚特兰大、达拉斯、休斯顿(有空调场馆) | 天气因素消除 |

**关键洞察**: 有空调/封闭屋顶的场馆完全消除天气影响，但露天场馆的极端条件可能改变比赛走势。

## 采集流程

### 赛前N天：搜索天气预报

对每场露天比赛（venue 在露天场馆的），用 web_search 获取赛前3-5天天气预报：

```bash
# 搜索模板
web_search("{city} June {date} weather forecast 2026")
web_search("{city} weather forecast {date}")
web_search("{venue/city} temperature humidity June {date}")
```

### 采集的数据要点

| 字段 | 说明 | 用途 |
|------|------|------|
| 🌡️ 气温(°C) | 比赛时间的预期温度 | 评估热应激 |
| 💧 湿度(%) | 相对湿度 | 体感温度+脱水速度 |
| 🌧️ 降水概率(%) | 下雨可能性 | 草皮湿滑/球速影响 |
| 💨 风速(km/h) | 风速 | 传中/长传准确性 |
| ☀️ 体感温度(°C) | 热指数/湿球温度 | 真实热应激指标 |
| 🕐 比赛时间 | 当地时间几点开球 | 日晒角度/夜间降温 |

### 体感温度评估标准

| 体感温度 | 级别 | 对比赛影响 |
|---------|------|-----------|
| <25°C | 🟢 舒适 | 无显著影响 |
| 25-30°C | 🟡 略热 | 需补水，影响轻微 |
| 30-35°C | 🟠 热 | 显著影响耐力，需轮换 |
| 35-40°C | 🔴 酷热 | 严重影响表现，补水暂停 |
| >40°C | 🆘 极端 | 可能触发医疗暂停/补水暂停 |

### 对非适应球队的影响评估

当体感温度 > 30°C 时，评估两队的热适应能力：

```
适应度 = 参赛国气候基线 + 近期训练环境 + 阵容来源联赛 + 历史热带比赛经验
```

具体基线数据见 `references/球队气候适应性分析.md`

## 特殊情景修正值

### 高温修正（露天场馆，体感温度 > 30°C）

| 温差程度 | 热适应方 vs 冷适应方 | 修正值 |
|---------|-------------------|--------|
| 体感30-35°C | 热适应 vs 冷适应 | -0.5（冷适应方） |
| 体感30-35°C | 热适应 vs 热适应 | 无修正 |
| 体感35-40°C | 热适应 vs 冷适应 | -0.7~1.0（冷适应方） |
| 体感35-40°C | 双方都不适应 | -0.3（双方） |
| 体感>40°C | 任何组合 | -0.5~1.0（非适应方更严重） |

### 高海拔修正

| 海拔 | 来自低海拔的球队 | 修正 |
|------|----------------|------|
| 2200m (墨西哥城) | 首次到高原比赛 | -0.5~0.8 |
| 1560m (瓜达拉哈拉) | 首次到高原比赛 | -0.3~0.5 |
| 540m (蒙特雷) | 任何球队 | -0.1（几乎无影响） |
