# 🏜️ 48支世界杯参赛队气候适应性分析

> 创建日期: 2026-06-06
> 目的: 为每场比赛提供两队的气候适应基线数据，用于天气因素修正

---

## 气候适应等级说明

| 等级 | 标识 | 含义 | 典型地区 |
|------|------|------|---------|
| 🟢 热带热适应 | 炎热环境下正常发挥 | 非洲/中东/加勒比/南美热带 |
| 🟢 亚热带热适应 | 可应对30-35°C湿热 | 美国南部/东亚夏季/南欧 |
| 🟡 温带适应当 | 在25-30°C正常 | 中欧/英国/南美温带 |
| 🟡 大陆性气候 | 对极端温差适应强 | 东欧/中亚/北美内陆 |
| 🔴 寒冷适应 | 在>25°C表现下降 | 北欧/斯堪的纳维亚 |
| 🔴 高海拔适应 | 在1500m+高原正常 | 安第斯国家 |

## 极高风险对位（高温比赛中最值得关注）

| 比赛场景 | 风险 | 预期影响 |
|---------|:----:|---------|
| 🏴󠁧󠁢󠁥󠁮󠁧󠁿 英格兰 vs 🟢 热适应队 | 🔴 极高 | 英格兰下半场体能崩盘概率高 |
| 🇳🇱 荷兰 vs 🟢 热适应队 | 🔴 极高 | 荷兰跑动距离下降15-20% |
| 🇧🇪 比利时 vs 🟢 热适应队 | 🔴 极高 | 比利时黄金一代体能劣势 |
| 🇸🇪 瑞典 vs 🟢 热适应队 | 🔴 极高 | **最极端温差** |
| 🇳🇴 挪威 vs 🟢 热适应队 | 🔴 极高 | **最极端温差** |
| 🇨🇭 瑞士 vs 任何队(高海拔场地) | 🟢 无风险 | 瑞士在高原反而有优势 |
| 🇧🇷 巴西 vs 🇸🇪 瑞典(蒙特勒露天) | 🔴 极高 | 最大温差对位 |
