# 日报图标与样式参考

> 用户明确要求：数据保留，但要配上解读，让普通人能看懂。
> 核心原则：**数据+解读+多角度**，每段数据后面跟一句人话总结。
> 参考模板看下方「完整报告案例」。

## 格式规则

1. **数据保留，后面跟人话解读**：每行数据后面用 `→` 或括号跟一句人话
2. 标题分隔线用 `─` 普通连字符（34个）
3. 第一块「一句话看懂今天」—— 2-3句话，涵盖：涨跌方向 + 板块特征 + 资金特征
4. 「🔥 板块热力图」—— 涨跌TOP3 + 一句总结
5. 「💰 聪明钱在干嘛」—— 北向+融资+龙虎榜机构 三信号对比 + 结论
6. 「📊 估值体检」—— 三个指数PE分位 + 人话解读
7. 「⚡ 今日异常」—— 有异常才写这节，没有就跳过
8. 每个板块的数据和解读之间空一格，但不要空太多行
9. 数据来源标注「东方财富妙想 + 理杏仁」
