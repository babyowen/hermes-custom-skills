# HotList Analyzer V6 精简版报告格式（Cronjob专用）

## 适用场景
Cronjob定时推送，需要避免消息被平台截断。总报告不超过2000字。

## 格式规范

| 元素 | 规范 |
|------|------|
| 标题行 | `📰 [热点标题]` |
| 元数据 | `热度: [X万] \| 跨[Y]平台` 或 `热度: [X万] \| [平台名]` |
| 分隔线 | `━━━━━━━━━━━━━━━━━━━━`（20个━） |
| 综述 | `【综述】[2句话说明是什么]` |
| 分析 | `【分析】` + 2-4个 `• [核心洞察]` 要点 |
| 建议 | `【建议】[1句行动建议]` |
| 总结区 | `💡 总结` + `🔴 立即关注:` + `🟡 持续跟踪:` + `✅ 数据来自11平台，2小时后更新` |

## 字数分配指导
- 总报告：≤2000字（目标1500字左右，留安全余量）
- 每条热点：300-400字（标题+热度+综述+3要点+建议）
- 总结区：≤150字
- 分隔线和格式符号：约200字
- **最多5条热点**，宁可分析3条完整的，不要7条被截断的

## 完整示例

```
🔥 全网热点简报
⏰ 生成时间: 2026-05-04 16:14
📊 数据: 11平台 291条 20个新热点

━━━━━━━━━━━━━━━━━━━━

📰 日本自民党前干事长：战争正靠近日本
热度: 1623.8万 | 今日头条

【综述】前首相石破茂公开警告现任首相高市早苗，若日本继续追随美国、挑衅中国，将成为下一场大战的前线。高市早苗正推动修订"安保三文件"，为长期战争做准备。

【分析】
• 日本政坛分裂：务实派vs右翼激进派，石破茂批高市是"美国代理人"
• 安保政策质变：从"专守防卫"转向"长期战争"准备，拟修订非核三原则
• 中方强烈反制：外交部质问日本是否重蹈军国主义覆辙，今年是东京审判80周年

【建议】密切关注日本安保政策修订动向及中日关系变化

━━━━━━━━━━━━━━━━━━━━

📰 南非警方安乐死野生巨鳄，肠道内发现人类遗骸
热度: 985万 | 跨2平台

【综述】南非59岁商人被洪水冲走后，警方在一条4.5米长、500公斤巨鳄肠道内发现人体遗骸，鳄鱼体内还有6双不同款式鞋子。

【分析】
• 极端天气风险：洪水导致人员失踪，反映气候变化下自然灾害频发
• 野生动物威胁：鳄鱼体内多双鞋子暗示可能多次攻击人类
• 救援手段升级：动用无人机、直升机精准定位，体现科技救援进步

【建议】关注极端天气预警，避免涉水危险区域

━━━━━━━━━━━━━━━━━━━━

💡 总结
🔴 立即关注: 日本安保政策修订及石破茂vs高市内斗
🟡 持续跟踪: 五一返程高峰交通数据、南非洪水后续
✅ 数据来自11平台，2小时后更新
```

## 选择标准（优先顺序）
1. 优先选择跨平台热点
2. 优先选择高热度新热点
3. 最多分析5条，确保内容精炼

## 字数控制原则

✅ **保留**：
- 热点标题
- 热度数据
- 1-2句话综述
- 2-3点核心分析（用 `•` 符号）
- 1句行动建议
- 总结建议

❌ **删除**：
- 详细的背景搜索过程
- 长篇的趋势预测
- 跨平台对比分析
- 历史数据对比
- 冗余的描述性文字

## 与V5模板的区别

| 维度 | V5（旧） | V6（当前） |
|------|---------|-----------|
| 分析符号 | `① ② ③` 圆圈数字 | `•` 项目符号 |
| 分析结构 | 单行连续 | 多行独立，更易读 |
| 综述长度 | 1句话 | 2句话，更完整 |
| 建议格式 | `【建议】[建议]` | 独立成行，清晰 |
| 分隔线 | 长短不一 | 固定20个━ |
