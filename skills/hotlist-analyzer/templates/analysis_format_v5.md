# HotList Analyzer V5 用户偏好分析格式

## 格式要求

用户偏好的分析结构（3个核心模块）：

```
📰 热点标题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【综述】
→ 新闻大概内容（让用户一眼知道这是什么新闻）

【热度】
→ 热度数据（各平台数据、排名、传播情况）

【深度分析】
→ 结合背景信息的分析（原因、影响、趋势等）
```

## 内容要求

1. **综述前置**：开头就要说清楚这是什么新闻
2. **热度独立**：单独一节说明传播情况
3. **分析聚焦**：只保留核心洞察，去除冗余内容
4. **具体分析**：要有具体细节，不是简单罗列

## 分析示例

### 示例1：政治类热点

```
📰 十项促进两岸交流合作措施发布
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【综述】
中央台办受权发布十项促进两岸交流合作的政策措施，涵盖：欢迎台湾同胞来大陆交流、推动两岸文化教育交流、促进两岸经济融合发展、便利台湾同胞在大陆生活、加强两岸青年交流、深化两岸媒体合作、扩大两岸民间交流、支持两岸共同弘扬中华文化、促进两岸基层交流、维护台湾同胞合法权益。

【热度】
• 抖音：1026万热度
• 腾讯新闻：313条评论，排名第2
• 跨2个平台传播，政治类严肃新闻获得高关注度

【深度分析】
1. 政策信号：大陆方面释放重大善意，显示推动两岸融合发展决心不变，在当前两岸关系背景下具有重要标志性意义。
2. 传播特点：严肃政治内容能在抖音获得千万级热度，说明两岸话题在普通民众中关注度高，也反映短视频平台对严肃内容的传播力。
3. 后续影响：预计会有实施细则出台，可能涉及签证便利、就业支持、教育优惠等，实际影响台湾同胞在大陆的工作生活。
4. 关注要点：需观察台湾方面和媒体反应，这是判断两岸关系走向的重要窗口，也可能影响两岸经贸、旅游、教育等领域。
```

### 示例2：社会类热点

```
📰 山西吕梁法官猥亵案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【综述】
山西吕梁市中院一法官被曝光猥亵案件当事人，目前警方已立案调查。事件引发社会广泛关注，涉及司法人员职业道德问题。

【热度】
• 澎湃新闻：第1位，1252万热度
• 腾讯新闻：第3位，892万热度
• 跨2个新闻传播平台，司法系统丑闻获得高关注

【深度分析】
1. 事件性质：司法系统内部人员涉嫌违法犯罪，严重损害司法公信力，属于权力监督失效的典型案例。
2. 传播爆发原因：①涉及权力人物（法官）②敏感议题（猥亵）③公众对司法公正的强烈关注④多平台同步传播形成共振
3. 后续发展预判：警方调查→司法处理→法院内部整顿→可能的系统性整改措施
4. 社会影响：可能引发公众对司法系统信任危机，但也体现舆论监督对权力制约的正面作用
```

## 两种版本的使用场景

| 版本 | 字数 | 适用场景 | 触发词 |
|------|------|---------|--------|
| **精简版** | <2000字 | Cronjob定时推送、避免截断 | "精简版热点分析"、"快速简报" |
| **深度版** | 3000-5000字 | 用户主动触发、详细解读 | "具体分析一下"、"深度分析" |

## 精简版格式（Cronjob用）

```
🔥 全网热点简报
⏰ 生成时间: 2026-04-12 16:30
📊 数据: 11平台 290条 20个新热点

━━━━━━━━━━━━━━━━━━━━

📰 十项促进两岸交流合作措施发布
热度: 1026万 | 跨2平台

【综述】中央台办发布十项措施，涵盖文化交流、经济合作等，释放重大善意信号。
【分析】1)政策层级高，涉及面广 2)多平台千万热度反映民众高关注 3)预计有实施细则出台

━━━━━━━━━━━━━━━━━━━━

📰 伊美伊斯兰堡谈判结束
热度: 912万 | 跨2平台

【综述】伊朗媒体称伊美在巴基斯坦首都的谈判已结束，内容未完全披露。
【分析】1)可能是试探性接触 2)地点选择显示谨慎 3)关注双方后续声明

━━━━━━━━━━━━━━━━━━━━

💡 总结
🔴 立即关注: 十项促进两岸交流合作措施发布
🟡 持续跟踪: 伊美伊斯兰堡谈判结束
✅ 2小时后更新
```

## 注意事项

- 分析要具体、有深度，不是简单罗列要点
- 要结合背景信息给出洞察，而不是泛泛而谈
- 热度数据要具体（多少万、排名第几、多少评论）
- 涉及多个平台时要说明差异（抖音侧重什么、新闻侧重什么）
- 精简版用于定时推送，深度版用于用户主动触发
