## 三、第三部分第七条 — 作文命题核心依据（原文+精解）

### 原文

> **"7. 持续深化考试内容和形式改革。** 以习近平新时代中国特色社会主义思想为指导，构建引导学生德智体美劳全面发展的考试内容体系。加强教考衔接，注重考查基础知识、基本技能、基本方法。**优化试题呈现方式和素材选取，融入科技前沿动态，浸润人文教育元素，加强项目式、探究式的真实情境问题设计，** 更好考查学生关键能力、学科素养和思维品质。"

### 逐句精解

| 原文表述 | 对作文命题的含义 | 实操指导 |
|:--------|:---------------|:---------|
| **融入科技前沿动态** | 作文素材应包含科技领域的最新进展（AI、航天、生物技术等），但不能纯科技叙事 | 热点库中保留DeepSeek/航天/量子等科技方向，但必须找到与人文的交汇点 |
| **浸润人文教育元素** | 科技素材不能孤立存在，必须"浸润"人文关怀——科技伦理、文化传承、人的价值 | 每个科技热点都要配套人文思辨角度（如"AI时代人的不可替代性""科技与传统文化的共生"） |
| **加强项目式、探究式的真实情境问题设计** | 作文题可能不再只是抽象思辨，而是设置具体情境、具体任务，让考生在情境中写作 | 关注"任务驱动型作文""情境交际型作文"的命题形式；材料更贴近真实生活场景 |
| **考查关键能力、学科素养和思维品质** | 区分度在思维深度，非套路模板；批判性思维、辩证思维是高分关键 | 预测方向必须有思辨空间，不能单一角度就能写好 |

### "三位一体"框架（2026年核心命题逻辑）

```
      ┌──────────────┐
      │  科技前沿动态  │
      │  (素材新维度)  │
      └──────┬───────┘
             │
    ┌────────┼────────┐
    │        │        │
    ▼        ▼        ▼
 ┌──────┐ ┌──────┐ ┌──────┐
 │ 人文  │ │ 真实  │ │ 思维  │
 │ 教育  │←│ 情境  │→│ 品质  │
 │ 元素  │ │ 设计  │ │ 考查  │
 └──────┘ └──────┘ └──────┘
             │
    ┌────────┼────────┐
    │        │        │
    ▼        ▼        ▼
 项目式    探究式    任务驱动
```

**核心要义**：科技 + 人文 + 真实情境，三者融合才是2026年命题的灵魂。

## 四、四条官方命题原则（通用，非仅语文）

| 原则 | 原文表述 | 对作文的含义 |
|:----|:--------|:------------|
| 🔴 **无价值不入题** | 所有试题必须体现立德树人 | 作文题一定有价值观导向，不可能纯技术讨论 |
| 🟠 **无思维不命题** | 重点考查逻辑思维、批判性思维和创新能力 | 套路文/模板文会大幅降分，思辨深度决定区分度 |
| 🟢 **无情境不成题** | 所有试题融入真实情境 | 纯抽象议论弱化，情境驱动型作文是趋势 |
| 🔵 **无任务不立题** | 每个试题有明确任务指向 | 作文题可能有更明确的任务指令，不仅是"写一篇文章" |

## 五、2026年作文方向全推演

### 5.1 可考性最高方向（直接对应通知原文）

| 优先级 | 方向 | 对应通知原文 | 理由 |
|:-----:|:----|:-----------|:-----|
| ⭐⭐⭐⭐⭐ | **科技与人文的交织** | "融入科技前沿动态，浸润人文教育元素" | 两项要求并列出现，说明命题组有意让科技与人文对话。这是2026年最大的新增考点 |
| ⭐⭐⭐⭐ | **真实情境下的思辨表达** | "加强项目式、探究式的真实情境问题设计" | 命题形式可能革新——给出具体场景任务，让考生在情境中完成写作 |
| ⭐⭐⭐⭐ | **文化传承中的创新** | 对应"人文教育元素"+"教考衔接" | 文化类是新高考I卷常青方向（2022围棋、2023故事、2025诗歌），但需融入时代创新 |
| ⭐⭐⭐ | **青年成长与时代选择** | 对应"立德树人"+"关键能力" | 经典方向，但需与新情境结合，避免老生常谈 |

### 5.2 科技+人文融合的具体角度预测

| 角度 | 思辨核心 | 谁能写好 | 公平性 |
|:----|:--------|:--------|:------|
| AI时代人的不可替代性 | 创造力/情感/价值观 vs 效率/计算 | 所有考生都有生活体验 | ✅ |
| 科技如何改变"传承" | 数字敦煌/AI修复古籍/方言保护 | 城市考生更熟悉但农村考生也能理解 | ✅ |
| 工具理性与价值理性 | 效率至上是否失落了"人" | 纯思辨，无需特定背景知识 | ✅ |
| "慢"在"快"时代的意义 | 专注/耐心/沉淀 vs 速度/效率 | 所有考生都能写 | ✅ |
| 技术的温度 | 科技能否解决老龄化/残疾人/偏远教育问题 | 有共情能力即可，无需技术背景 | ✅ |

### 5.3 低概率方向（与通知精神不符）

| 方向 | 排除理由 |
|:----|:---------|
| 纯政治表态式命题 | 违背"反宿构""个性化表达"要求，也违背"无思维不命题" |
| 纯科技叙事（AI万能论） | 违背"浸润人文教育元素"要求，科技必须与人文结合 |
| 过于抽象脱离现实 | 违背"真实情境问题设计"要求 |
| 纯抒情/纯叙事无思辨 | 违背"考查思维品质"要求 |

## 六、与2025年命题方向的延续与变化

| 维度 | 2025年通知 | 2026年通知 | 变化 |
|:----|:----------|:----------|:-----|
| 核心表述 | 未明确提及科技+人文融合 | **明确写入"融入科技前沿动态，浸润人文教育元素"** | 🆕 首次并列，成为命题核心方向 |
| 真实情境 | 提及"情境设计" | **升级为"项目式、探究式的真实情境问题设计"** | 🆕 增加了"项目式""探究式"两个定语 |
| 教考衔接 | 提及 | **保留并强化** | 延续 |
| 立德树人 | 核心 | **核心（"无价值不入题"）** | 延续 |

### 关键洞察

2026年通知与2025年最大的不同是**首次将"科技前沿动态"和"人文教育元素"并列写入命题要求**。这不是巧合——这是命题组对时代命题的回应：AI时代到了，高考作文必须让学生思考科技与人的关系，但具体角度不会是简单的"AI利弊谈"，而是更精巧、更富思辨性的切入。

## 七、配套政策的信号强化 — 5部门《"人工智能+教育"行动计划》

> 2026年5月13日，教育部、发改委、工信部、科技部、国家数据局联合印发《"人工智能+教育"行动计划》。
> 这是教学〔2026〕1号的**具体落地方案**，不是重复，而是从"命题要求"到"教育体系变革"的延伸。

### 6.1 对作文预测的直接意义

| 原文表述 | 对应教学〔2026〕1号 | 作文含义 |
|:--------|:------------------|:---------|
| **"坚持科技教育与人文教育相结合"** | "融入科技前沿动态+浸润人文教育元素" | 进一步确认科技与人文的并列关系，不是二选一 |
| **"项目式、探究式、场景式育人"** | "加强项目式、探究式的真实情境问题设计" | "场景式"是新增定语 → 真实情境作文概率进一步提升 |
| **"注重学生启智、心灵的培养"** | "考查关键能力、学科素养和思维品质" | 思辨性仍为核心，启发智慧+滋养心灵并重 |
| **"引导学生科学认识、合理利用智能技术"** | "融入科技前沿动态" | 从"知道"到"会用"到"会思考"的递进 |

### 6.2 关键金句提取（可直接引用到预测题/素材）

> "**坚持科技教育与人文教育相结合**，注重学生的启智、心灵的培养，引导学生科学认识、合理利用智能技术，提升学生智能素养，激发学生好奇心，培养创新思维，提高认知思考和解决复杂问题的能力。"

> "**打造项目式、探究式、场景式育人模式**，引导学生学会思考，培养胜任智能时代的能力。"

### 6.3 实操建议

- 该行动计划是教学〔2026〕1号的**配套政策**，在热点库中标注为 **"5部门联合——《"人工智能+教育"行动计划（2026.5.13）"**
- 作为教育部方向的"政策回声"，可在预测报告中的"教育部方向匹配度"论证中引用
- 注意：该计划本身是教育政策层面的文件，不是命题直接材料，但**强化了"科技+人文"的可考性论证**
